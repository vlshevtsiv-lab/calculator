# calculator

Tkinter Calculator
Description

This project is a simple GUI calculator built with Python and Tkinter. It supports basic arithmetic operations (+, -, *, /) and includes a theme system allowing users to switch between light, dark, blue, green, and red themes. The application demonstrates GUI design, event handling, and dynamic styling in Python.

Features

Perform basic arithmetic calculations

Clear (C) and evaluate (=) expressions

Dynamic input display with real-time updates

Multiple themes: Light, Dark, Blue, Green, Red

Simple and intuitive GUI layout

How to Run

Make sure Python 3 is installed.

Save the script as calculator.py.

Run the program:

python calculator.py


Use the buttons to input numbers and operations.

Use the menu to switch between color themes.

Learning Goals

Build GUIs using Tkinter

Handle button events and input validation

Work with dynamic widget styling

Implement a theme-switching system
